# 06 — TL;DR (Pareto: 5 habits)

If you only do five things from this entire system, do these. Together they capture ~80% of the achievable outcome.

---

## 1. Sleep ≥7h, ≥5 nights/week

**Why**: Below this, you cognitively perform like a sleep-deprived person *without noticing*. ([Van Dongen 2003](https://pubmed.ncbi.nlm.nih.gov/12683469/))

**How**: Bedtime alarm at T-30min before lights out. Caffeine cutoff 8h before bed. No interactive screens 30 min before bed.

**Track in**: `/energy` → Sleep tab.

---

## 2. One 90-min deep work block before noon, ≥4 days/week

**Why**: This is where career-defining output happens. Newport, PG, Karpathy all converge on this. Everything else is shallow.

**How**: Calendar block. Notifications off. One project per block. Use the `DeepWorkTimer` on `/attention`.

**Track in**: `/attention`.

---

## 3. 3×/week × 40 min Zone-2 cardio

**Why**: Strongest non-sleep cognitive intervention with peer-reviewed evidence. ([Erickson 2011 PNAS](https://www.pnas.org/doi/10.1073/pnas.1015950108), [Northey 2018 BJSM](https://pubmed.ncbi.nlm.nih.gov/28438770/))

**How**: Walk briskly, jog, cycle, swim. Conversational pace. 40 min. Not optional 3 days/week.

**Track in**: `/energy` → Exercise tab.

---

## 4. MIT (Most Important Task) before email/Slack each morning

**Why**: Whatever you do first, you do best. Email-first means you spend your peak hour on other people's priorities.

**How**: Set tomorrow's MIT during the daily review (`/review/daily`). Tomorrow morning: do it before opening any communication tool.

**Track in**: Daily Dashboard (`/`).

---

## 5. Sunday 30-min weekly review

**Why**: The system catches your slips before they compound for weeks. Without this, the entire system rots in a month.

**How**: Open `/review/weekly` Sunday 17:00. Review last week stats. Set top-3 outcomes for next week. Decide if any project moves Active ↔ Maintenance ↔ Paused.

**Track in**: `/review/weekly`.

---

## What this list excludes (deliberately)

- Meditation, journaling, gratitude practice → fine if they help you, but not load-bearing for the core outcome.
- Cold showers → see [`05-anti-patterns.md`](./05-anti-patterns.md).
- Detailed habit stacking (10 micro-habits) → adds friction; the 5 above are enough.
- Reading a book a week → see anti-pattern #9.
- Tracking calories/macros → not productivity-related.

---

## The order of operations when something goes wrong

If you can only fix one thing at a time:

1. **Fix sleep first.** Everything downstream of sleep gets worse if sleep breaks.
2. **Then deep work.** Once sleep is OK, restore the morning block.
3. **Then exercise.** Easier to add when the first two are stable.
4. **Then MIT discipline.** Once you have hours, focus them.
5. **Then weekly review.** This is the *governance* layer; restore once the daily layer is stable.
