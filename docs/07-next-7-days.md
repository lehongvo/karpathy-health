# 07 — Next 7 Days Action Plan

A concrete week-1 roll-out. Don't try to adopt all of `06-tldr.md` at once — adopt one habit per day. By Day 7, the system is running.

---

## Day 1 (Monday) — Sleep baseline

- [ ] Set bedtime alarm for 22:30 (or whatever T-30min before your target sleep is).
- [ ] Set a hard caffeine cutoff: no caffeine after 14:00. (Adjust if your bedtime is earlier/later.)
- [ ] Tonight: log your sleep on `/energy` → Sleep.
- [ ] **No new habits today.** Just sleep.

---

## Day 2 (Tuesday) — Deep work baseline

- [ ] Open calendar. Block **08:30-10:00** as "Deep Work — [pick one project]".
- [ ] Tomorrow morning: open the dashboard, hit "Start Deep Work Timer" on `/attention`. One project. Notifications off.
- [ ] Tonight: log sleep again.

---

## Day 3 (Wednesday) — Notification audit

- [ ] Run the notification hygiene checklist in `03-attention-protocol.md` §D.
- [ ] Phone Focus mode: 06:00-12:00 daily.
- [ ] Slack: notifications off except DMs + @mentions.
- [ ] Do today's deep work block. Log distraction count.

---

## Day 4 (Thursday) — Exercise restart

- [ ] **30-min walk today.** That's it. Don't try to do more.
- [ ] Log it on `/energy` → Exercise.
- [ ] Do today's deep work block.

---

## Day 5 (Friday) — Project triage

- [ ] Open `/tasks`.
- [ ] List every project / responsibility you currently hold.
- [ ] Sort into: **Active (max 2)** / Maintenance / Paused.
- [ ] If you have >2 in Active: move one. Force the decision.
- [ ] If something gets paused, send the stakeholder one short message. ("I'm pausing work on X until [date]. Will check in then.")

---

## Day 6 (Saturday) — Catch-up + 1st cardio session

- [ ] **40-min Zone-2 cardio.** Conversational pace. Outside if possible (sunlight bonus).
- [ ] No deep work expectations today. Recovery + 1 hour of low-pressure Rust reading is fine.
- [ ] Log sleep, exercise.

---

## Day 7 (Sunday) — First weekly review

- [ ] 17:00 — open `/review/weekly`.
- [ ] Look at this week's stats:
  - Sleep avg: ___
  - Deep work blocks completed: ___ / 4 target
  - Exercise sessions: ___ / 3 target
  - MITs hit: ___ / 5
- [ ] Top 3 outcomes for next week (not 10, **3**).
- [ ] Adjust calendar for next week's deep work blocks.

---

## Week 2 onward

You're now running the loop. Reference:
- `02-energy-protocol.md` for sleep/exercise/nutrition rules
- `03-attention-protocol.md` for deep work
- `04-failure-recovery.md` when something slips
- `06-tldr.md` if you forget the core 5 habits

**Don't add anything new for at least 2 weeks** after Day 7. Let the base loop settle.

---

## What success looks like at Day 30

| Metric | Target | Why |
|---|---|---|
| Sleep avg | ≥7h, ≥21 nights of 30 | Cognitive baseline |
| Deep work blocks | ≥16 / 30 days (~4/wk) | Output engine |
| Exercise sessions | ≥12 / 30 days | BDNF + mood baseline |
| MITs hit rate | ≥70% | Focus discipline |
| Weekly reviews | 4 / 4 | System governance |

If you hit those, the system works. Iterate. If you're below 50% on any of them, the issue isn't the system — it's a constraint we haven't surfaced. Open `decision-log.md` and write what's actually blocking you.
